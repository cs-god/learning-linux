# 变量赋值

**语法规则**：`变量名=变量内容 ` <font color=FF0000>(等号两边不能直接连接空格)</font>

**单引号和双引号的区别**

双引号内的特殊字符如$等，可以保持原有的特性。

```bash
var="lang is $LANG"    # lang is zh_CN.UTF-8
```

单引号内的特殊字符仅为一般字符（纯文本）。

```bash
var='lang is $LANG'    lang is $LANG
```

##### 2.2 变量使用

**语法规则**：`${变量名}`

最好使用"${变量名}"，防止出现字符串中出现空格的情况。

##### 2.3 变量取消

**语法规则**：`unset 变量名称`

### 3、case...esac

```bash
case $变量名称 in
	"第一个变量内容")
		程序段
	;;	
	"第二个变量内容")
		程序段
	;;	
	*）
		程序段
		exit 1
		;;
esac		
```
